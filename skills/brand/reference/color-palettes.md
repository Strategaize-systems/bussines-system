## Verwendungshinweise

- **Primary** = Hauptfarbe, Logo, Navigation, CTAs
- **Secondary** = Unterstützend, Hover-States, Links
- **Accent** = Hervorhebungen, Erfolg, besondere Elemente
- **Neutral** = Hintergründe (hell) / Text (dunkel)

Paletten können als Ausgangspunkt dienen. Der Skill passt Farben an die Eingaben der Firma an.
