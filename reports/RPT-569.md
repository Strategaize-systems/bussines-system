### Pfad 1 — XSS-Probe-1 (script-tag) — ✅ PASS

**Input:**
```html
<p>Hallo, harmlos.</p><script>document.body.innerHTML = "PWNED-SCRIPT-TAG";</script><p>Tschuess.</p>
```

**Result (via DOMParser-Inspect des iframe.srcdoc):**
- iframe `sandbox=""` exakt ✅
- `srcdocContainsScript: false` — DOMPurify hat `<script>`-Tag komplett entfernt ✅
- `srcdocContainsPwned: false` ✅
- `pageBodyContainsPwned: false` — kein DOM-Bypass ✅
- `pageDocumentTitle: "Business Cockpit — Strategaize"` — kein Title-Hijack ✅
- Body rendert: `<p>Hallo, harmlos.</p><p>Tschuess.</p>` ✅
- Console: 0 Errors, 0 Warnings ✅

### Pfad 2 — XSS-Probe-2 (svg + onerror + javascript:) — ✅ PASS

**Input:**
```html
<p>Vor dem Angriff.</p>
<svg onload="document.title='PWNED-SVG'"></svg>
<img src=x onerror="document.title='PWNED-IMG'">
<a href="javascript:document.title='PWNED-A'">Klick hier</a>
<p>Nach dem Angriff.</p>
```

**Result:**
- `<svg>` komplett entfernt (FORBID_TAG) ✅
- `<img>` bleibt, aber `src=x` + `onerror=...` entfernt (FORBID_ATTR + URL-Scheme-Filter) ✅
- `<a>Klick hier</a>` bleibt, aber `href="javascript:..."` ist `null` (ALLOWED_URI_REGEXP) ✅
- bodyHTML: `<p>Vor dem Angriff.</p><img><a>Klick hier</a><p>Nach dem Angriff.</p>` ✅
- Console: 0 Errors, 0 Warnings ✅

### Pfad 3 — Echte HTML-Mail (IONOS Welcome) — ✅ PASS
