### Was neu gelernt wurde

#### Learning #1 тАФ ISSUE-071 systemd-timer-Pattern fuer Self-Hosted-Coolify

**Vorher**: ISSUE-071 war seit 4 Auftritten (14./16./18./19. Mai) als "weekly Cron User-Pflicht-Action" dokumentiert, aber nie eingerichtet. Coolify-Auto-Cleanup hat 3x das Risiko gemildert, aber das war Glueck, nicht Design.

**Jetzt**: systemd-timer auf 91.98.20.191 eingerichtet:
- `/etc/systemd/system/docker-prune.service` (ExecStart: `docker builder prune -af` + `docker image prune -af`)
- `/etc/systemd/system/docker-prune.timer` (OnCalendar=Sun 03:00 + RandomizedDelaySec=600 + Persistent=true)
- `systemctl enable --now docker-prune.timer`
- Test-Lauf sofort: 722.4MB reclaimed, Disk 88%тЖТ82%, exit 0/SUCCESS

**Pattern fuer Dev-System**: systemd-timer auf Hetzner-Host ist sauberer als Coolify-Cron-Container (kein Container-Overhead, keine Docker-Socket-Mount-Permissions, Hetzner-Reboot-resilient via Persistent=true). Sollte als Default-Pattern fuer alle Strategaize-Coolify-Hetzner-Deployments dokumentiert werden (kommt in V7.7-Skill-Improvement).

#### Learning #2 тАФ Playwright-MCP-Limitierung fuer dnd-kit horizontal-scrolled Containers

**Beobachtung**: V8 SLC-813 AC11 (Drag-Drop von Deal-Card auf Verloren-Column) ist via Playwright-MCP nicht zuverlaessig testbar тАФ Pipeline-View hat 12 Stages (3280px breit) bei Viewport 605px. Source-Card und Drop-Target koennen nicht gleichzeitig im Viewport sein. dnd-kit `activationConstraint: distance: 5` benoetigt PointerEvent-Sequenz die MCP-Drag-Tool nicht zuverlaessig emuliert.

**Workaround in V8**: Backend-Pfad via 42 Unit-Tests verifiziert (RPT-484), Live-Bedrock-Region-Pfad via Tagesanalyse-Click verifiziert (gleicher Bedrock-Client). Manual-User-Smoke-Skript dokumentiert (30 Sek, optional).

**Mitigation-Optionen fuer kuenftige Smokes** (V7.7-IMP-Kandidat):
- Viewport-Resize auf 3500px (mcp__playwright__browser_resize) vor Drag-Drop-Tests
- Headless-Mode mit grosser Viewport-Aufloesung als Default fuer Pipeline-View
- Alternative: Server-Action-Direct-Test ueber Browser-Konsole (statt UI-Drag)
