## 3. Findings

### Blocker

- **F1: Meeting-Dropdown-Click crasht Deal-Detail-Page komplett** in
  [deal-action-bar.tsx:169-206](cockpit/src/components/deals/deal-action-bar.tsx#L169-L206)
  Console-Errors: `Base UI error #31` + `React Minified error #418` (Hydration).
  - **Root-Cause:** `<MeetingSheet trigger={<DropdownMenuItem>}>` wird als
    direktes Kind von `<DropdownMenuContent>` gerendert. MeetingSheet returnt
    intern eine `<Sheet>` mit `<SheetTrigger>{trigger}</SheetTrigger>` — das
    macht `<Sheet>` zum direkten Kind von `<DropdownMenuContent>`, was Base UI
    nicht akzeptiert (DropdownMenuContent erwartet nur DropdownMenuItem /
    DropdownMenuLabel / DropdownMenuSeparator als direkte Kinder).
  - **Impact:** AC3 (Meeting-Dropdown mit "Termin planen" + "Sofort starten")
    ist UNERREICHBAR — User kommt zu keinem der beiden Optionen, Page haengt
    auf Error-Screen "This page couldn't load" bis Reload.
  - **Repro:** Live verifiziert 2026-05-10 11:46 + 11:48 UTC+2 (zwei
    unabhaengige Reloads, beide Male reproduzierbar, full-page-crash).

- **F2: Mehr-Aktionen-Dropdown-Click crasht Deal-Detail-Page komplett** in
  [deal-action-bar.tsx:240-295](cockpit/src/components/deals/deal-action-bar.tsx#L240-L295)
  Gleicher Console-Error wie F1.
  - **Root-Cause:** Mindestens zwei unzulaessige direkte Kinder von
    `<DropdownMenuContent>`:
    1. `<div className="md:hidden">…</div>` Wrapper um DropdownMenuItem +
       DropdownMenuSeparator (line 257-263)
    2. Conditional-Render `{deal.contact_id && (...)}` (line 288-293) mit
       inline-`<DropdownMenuItem>` — moeglicherweise OK, aber `<EnrollButton
       trigger={<DropdownMenuItem>}>` (line 265) wrappt extern in `<span
       onClick>` was als Kind von DropdownMenuContent ebenfalls invalid sein
       koennte (siehe enroll-button.tsx:64).
  - **Impact:** AC4 (Mehr-Menue mit Cadence + Workflow-Trigger +
    Activity-Vollformular + Kontakt + Mobile-Angebot) ist UNERREICHBAR.
  - **Repro:** Live verifiziert 2026-05-10 11:50 UTC+2 (eigenstaendiger Click
    auf nur Mehr-Aktionen, ohne vorherigen Meeting-Click).
