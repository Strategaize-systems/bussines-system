## Architecture Alignment

- **Cron-API-Routes** (DEC-033): Implementiert als Next.js API Routes mit x-cron-secret Protection
- **IMAP-Sync** (FEAT-405): ImapFlow + Mailparser, inkrementell via UID-Tracking, max 50/Sync (500 initial)
- **Thread Detection**: 3-Stufen (In-Reply-To → References → Normalized Subject + Contact + 14-Tage-Fenster)
- **Contact Matching**: Exact → Domain → null (Public Domains ausgeschlossen)
- **Retention**: 90 Tage Default (IMAP_RETENTION_DAYS konfigurierbar)
- **Admin Client**: Cron-Routes nutzen createAdminClient() (service_role, keine User-Session)
