## Migration Details

1. **activities** — +source_type (TEXT), +source_id (UUID) fuer Rueckverlinkung
2. **profiles** — role-Default von 'owner' auf 'admin' geaendert, bestehende Werte migriert, +team (TEXT, nullable)
3. **meetings** — Neue Tabelle (16 Spalten: title, scheduled_at, duration, location, participants, agenda, outcome, notes, transcript, deal/contact/company FKs, status, created_by)
4. **calendar_events** — Neue Tabelle (13 Spalten: title, start/end_time, type, description, location, deal/contact/company/meeting FKs, created_by)
5. **audit_log** — Neue Tabelle (8 Spalten: actor_id, action, entity_type, entity_id, changes JSONB, context)
6. **9 Indexes** — idx_activities_source (partiell), 3x meetings, 2x calendar_events, 3x audit_log
7. **RLS** — authenticated_full_access auf allen 3 neuen Tabellen (V1-Muster, MIG-006 baut um)
8. **Grants** — authenticated + service_role auf allen 3 neuen Tabellen
