   rendert") requires the real Mein-Tag capture.

   credentials available in this session; no local dev server started

   These ACs need to be verified during `/qa SLC-881`.
3. **Vitest mock pattern needed default-export.** `vi.mock("node:fs/promises", () => ({ readFile }))`
   failed under Vitest 4 because the loader's tsc-compiled output also
   reaches for `default`. Worked around with explicit

   for the dev-system feedback memory if the issue recurs.
