```

    ↓ (Cron: meeting-summary / classify)
signal_status = 'pending' auf meetings / email_messages
    ↓ (Cron: signal-extract, alle 5 Min)
extractSignals(text, dealContext, sourceType, sourceId)
    ↓ (Bedrock Claude + Zod-Parsing + Confidence-Filter)
ai_action_queue INSERT (source=signal_meeting/email, proposed_changes JSONB)
    ↓ (Server: getPendingInsights)
Mein Tag → InsightSuggestions → PropertyChangeCard
    ↓ (User: Approve/Reject/Batch)
approveInsightAction → applyProposedChange → Deal UPDATE
    ↓ (Audit: Activity INSERT type=ai_applied)
Deal-Workspace → DealHeader → KI-Badge (30 Tage)
```

```

    ↓ (POST /api/signals/extract, Auth + Rate-Limit)

    ↓ (gleicher Pfad wie oben)
ai_action_queue → Mein Tag → Approve → Badge
```
